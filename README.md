# COVID19TID
COVID19TID is a web application for traffic inflation data for COVID-19 dispersion control.

## Technologies
  * HTML5
  * CSS3
  * Javascript
  * PHP
  * SQL
  * Leaflet.js
  * Chart.js
  * Font Awesome Icons
  * jQuery
  * Ajax
